Personal Portfolio Website (Prototype)

Tentang Proyek | About the Project

Website ini merupakan website portofolio pribadi yang saat ini masih berada dalam tahap pengembangan awal. Fokus utama pengembangan saat ini adalah pada front-end design, termasuk tata letak, struktur halaman, dan pengalaman pengguna (UI/UX).

Saat ini website masih berfungsi sebagai prototipe desain dan belum memiliki implementasi backend. Beberapa fitur belum tersedia atau masih menggunakan konten sementara (placeholder).

This project is a personal portfolio website that is currently in the early development stage. The current focus is on front-end design, including layout, page structure, and user experience (UI/UX).

At this stage, the website acts as a design prototype and does not yet include backend implementation. Some features are incomplete or use placeholder content.
